# 📂 Proje #5 — Yüz Algılama Uygulaması

OpenCV kütüphanesi ve önceden eğitilmiş Haar Cascade modellerini kullanarak fotoğraflardaki insan yüzlerini tespit eden ve işaretleyen masaüstü aracıdır.

## 📄 Klasör İçeriği
- `yuz_algilama.py`: Tkinter tabanlı yüz algılama arayüz kodu.
- `face_detector.xml`: Yüz tespiti için kullanılan Haar Cascade modeli.
- `yuz_algilama_Aciklamalari.ipynb`: OpenCV ile görüntü işleme temellerini anlatan notebook.

## 📦 Gerekli Kütüphaneler
- `opencv-python`
- `numpy`
- `Pillow`

## 🚀 Çalıştırma
```bash
python yuz_algilama.py
```
