# 📂 Proje #8 — Link Kısaltma Uygulaması

TinyURL API'sini kullanarak uzun web bağlantılarını hızlıca kısaltan ve tek tıkla işletim sistemi panosuna (clipboard) kopyalayan pratik bir araçtır.

## 📄 Klasör İçeriği
- `link_kisaltma.py`: Link kısaltma ana kod dosyası.
- `link_kisaltma_Aciklamalari.ipynb`: API istekleri ve pano kopyalama mantığını anlatan notebook dosyası.

## 📦 Gerekli Kütüphaneler
- `requests`
- `pyperclip`

## 🚀 Çalıştırma
```bash
python link_kisaltma.py
```
