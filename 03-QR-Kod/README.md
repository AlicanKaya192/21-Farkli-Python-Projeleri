# 📂 Proje #3 — QR Kod Oluşturucu

Tkinter arayüzü ile girilen bir metin ya da URL'yi QR koda dönüştüren ve bunu SVG olarak ölçeklenebilir formatta kaydeden masaüstü uygulamasıdır.

## 📄 Klasör İçeriği
- `3.1_qr_kod.py`: Arayüzlü QR kod oluşturucu ana kodu.
- `QR_Kod_Aciklamalari.ipynb`: Karekod üretim mantığını açıklayan notebook dosyası.

## 📦 Gerekli Kütüphaneler
- `pyqrcode`
- `Pillow`

## 🚀 Çalıştırma
```bash
python 3.1_qr_kod.py
```
