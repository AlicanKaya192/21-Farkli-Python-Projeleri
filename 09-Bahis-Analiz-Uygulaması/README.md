# 📂 Proje #9 — Bahis Analiz Uygulaması

Sporx üzerinden takımların fikstür verilerini çekerek, Poisson dağılımı ve Form Decay (zaman ağırlıklı ortalama) algoritmaları yardımıyla maç gol ve derbi tahminleri üreten analiz aracıdır.

## 📄 Klasör İçeriği
- `bahis_analiz.py`: Poisson modeliyle çalışan tahmin uygulaması ana kodu.
- `bahis_analiz_Aciklamalari.ipynb`: Poisson olasılık dağılımını ve veri çekmeyi açıklayan notebook.

## 📦 Gerekli Kütüphaneler
- `requests`
- `beautifulsoup4`

## 🚀 Çalıştırma
```bash
python bahis_analiz.py
```
