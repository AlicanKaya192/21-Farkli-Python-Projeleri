# 📂 Proje #2 — Dijital Masaüstü Saati

Tkinter kütüphanesi ile oluşturulmuş, anlık olarak güncellenen sade ve şık bir masaüstü dijital saat uygulamasıdır.

## 📄 Klasör İçeriği
- `2.1_dijital_saat.py`: Masaüstü saat uygulaması ana kodu.
- `Dijital_Saat_Aciklamalari.ipynb`: Uygulamanın geliştirilme aşamalarını anlatan notebook dosyası.

## 📦 Gerekli Kütüphaneler
- Yerleşik `tkinter` ve `time` modülleri kullanılır, harici kütüphane gerekmez.

## 🚀 Çalıştırma
```bash
python 2.1_dijital_saat.py
```
