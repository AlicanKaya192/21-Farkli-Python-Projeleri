# 📂 Proje #6 — Instagram Gönderi İndirici Bot

Instaloader kütüphanesi kullanılarak belirtilen bir Instagram hesabının profilindeki gönderileri (resim/video) otomatik olarak bilgisayara indiren masaüstü botudur.

## 📄 Klasör İçeriği
- `insta.py`: Gönderileri indiren Tkinter arayüzlü ana kod.
- `insta_Aciklamalari.ipynb`: Instaloader kullanımı ve arayüz entegrasyonu rehberi.

## 📦 Gerekli Kütüphaneler
- `instaloader`

## 🚀 Çalıştırma
```bash
python insta.py
```
