# 📂 Proje #10 — Instagram Geri Takip Etmeyenler Uygulaması

Instagram'dan indirilen takipçi ve takip edilen verilerini karşılaştırarak sizi geri takip etmeyen kişileri tespit eden, sonuçları Excel veya PDF olarak raporlayabilen çok iş parçacıklı (multithreaded) bir masaüstü uygulamasıdır.

## 📄 Klasör İçeriği
- `geri_takip_etmeyenler.py`: Takipçileri karşılaştırıp raporlayan ana kod.
- `geri_takip_etmeyenler_Aciklamalari.ipynb`: Küme işlemleri ve threading yapısını anlatan notebook dosyası.

## 📦 Gerekli Kütüphaneler
- `pandas`
- `openpyxl`
- `fpdf`
- `beautifulsoup4`

## 🚀 Çalıştırma
```bash
python geri_takip_etmeyenler.py
```
