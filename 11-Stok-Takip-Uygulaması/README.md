# 📂 Proje #11 — Stok Takip Uygulaması

SQLite veritabanı altyapısını kullanarak ürün stok girişlerini, adetlerini, birim fiyatlarını ve toplam stok değerlerini yerel olarak saklayan ve yöneten Tkinter masaüstü uygulamasıdır.

## 📄 Klasör İçeriği
- `stok_takip.py`: SQLite veritabanı entegrasyonuna sahip stok takip ana kodu.

## 📦 Gerekli Kütüphaneler
- Yerleşik `sqlite3` ve `tkinter` kütüphanelerini kullanır. Ekstra kurulum gerekmez.

## 🚀 Çalıştırma
```bash
python stok_takip.py
```
