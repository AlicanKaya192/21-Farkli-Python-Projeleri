# 📂 Proje #1 — Web Sitelerinden Veri Çekme

Web scraping (web kazıma) tekniklerini öğrenmek için hazırlanmış üç aşamalı veri çekme projesidir.

## 📄 Klasör İçeriği
- `1.1_web.py`: Temel web scraping mantığı. HTTP GET istekleri gönderme ve HTML elementlerini parse etme.
- `1.1_web_Aciklamalari.ipynb`: Temel web scraping adımlarını açıklayan rehber.
- `1.2_korona_veri.py`: Sağlık Bakanlığı Covid-19 verilerini tablolardan çekme uygulaması.
- `1.2_korona_veri_Aciklamalari.ipynb`: Tablo verisi ayıklama mantığını açıklayan eğitim dosyası.
- `1.3_itopya_islemci_fiyat_listesi_cekme.py`: İtopya sitesinden işlemci ürün adları, fiyatları ve linklerini çeken e-ticaret botu.
- `1.3_itopya_islemci_fiyat_listesi_cekme_Aciklamalari.ipynb`: E-ticaret sitelerinden gelişmiş veri çekme rehberi.

## 📦 Gerekli Kütüphaneler
- `requests`
- `beautifulsoup4`

## 🚀 Çalıştırma
```bash
python 1.1_web.py
python 1.2_korona_veri.py
python 1.3_itopya_islemci_fiyat_listesi_cekme.py
```
